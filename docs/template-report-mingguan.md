# Template Report Mingguan (Index)

Template report mingguan sudah dipisahkan menjadi dua dokumen agar lebih rapi:

1. Progress Report Mingguan
   - File: [template-progress-report-mingguan.md](template-progress-report-mingguan.md)
   - Fokus: capaian hasil kerja, KPI, target minggu depan.

2. Development Process Mingguan
   - File: [template-development-process-mingguan.md](template-development-process-mingguan.md)
   - Fokus: proses engineering (planning, code review, testing, deployment, blocker).

## Saran Penggunaan

1. Isi dua dokumen ini setiap akhir minggu.
2. Simpan hasil report dengan penamaan tanggal.
3. Lampirkan link PR, board task, dan screenshot demo agar audit trail lengkap.
